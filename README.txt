# Semi-Supervised-Self-Organised-Prototype-Tree-Model-for-Few-Shot-Learning
The Python code for the research paper titled "A semi-supervised self-organised prototype tree-based method for few-shot remote sensing scene classification".
This work is described in:
============================================================================================================================
Gu, X., Kerim, A., Zhang, C., Han, J., Atkinson, P. M., and Shen, Q. (2026). A semi-supervised self-organised prototype 
tree-based method for few-shot remote sensing scene classification. Knowledge-Based Systems, 347, 116350.
